================
Constant Editor
================

.. include:: ../../Includes.txt

In addition to the FlexForm, many options can be set from the Constant
Editor. Indeed, there are so many options that the Constant Editor has
been divided into three forms:

- Calendar Base (General)

- Calendar Base (Templates)

- Calendar Base (RSS)
