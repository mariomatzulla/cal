=================================
Step 5: Add the Static Templates
=================================

.. include:: ../../Includes.txt

In order for the Calendar Base extension to appear in your Fronted, you will need to two Static Templates to the Extension Template Record (alternatively, they can be installed in your site's main Template Record). The two Static Templates required by the Calendar Base extension are:

- Standard CSS-based template (cal)

- Standard CSS styles (cal)


Please refer also to the chapter ":ref:`TemplateFilesAndMarkers`" to learn more about the calendar templates.
